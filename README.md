# LiveCasino.Reviews

Independent live dealer casino reviews, live blackjack & poker guides, payment
comparisons and responsible-gambling resources — built as a fast, SEO-focused
editorial platform.

**Live positioning:** _Independent live casino reviews, live blackjack guides, poker game
explanations, payment comparisons, and safe gambling resources for players who want to
compare live dealer sites before signing up._

---

## Tech stack

- **Next.js 15** (App Router) + **React 19**, TypeScript
- **next/font** (Plus Jakarta Sans + Inter) — self-hosted, zero layout shift
- Fully static export (SSG) — 88+ prerendered pages
- Custom CSS design system (no UI framework) — see [`DESIGN.md`](./DESIGN.md)

## Getting started

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build (static)
npm start        # serve the production build
```

## Design

The UI was modernised by analysing **casino.org** and adopting its typography, colour
language (violet primary / emerald CTA / gold ratings), column mega-menu and trust-first
layout. Full rationale — including why the mega-menu was chosen — is in
[`DESIGN.md`](./DESIGN.md).

## Structure

```
src/
  app/                 App Router pages + dynamic routes ([slug])
    reviews/[slug]/    individual casino reviews
    live-blackjack/    hub + article routes  (and poker, games, payments, safety, bonuses, providers)
    countries/[slug]/  country guides
    sitemap.ts         XML sitemap        robots.ts   robots.txt
  components/          Header (mega-menu), Footer, cards, tables, FAQ, schema, ArticleView…
  data/                casinos, articles registry, providers, countries, navigation
  lib/                 content engine, schema.org helpers, site config, route map
```

## SEO

Per-page titles/meta/canonical, JSON-LD (`Organization`, `WebSite`, `Article`, `Review`,
`FAQPage`, `BreadcrumbList`), breadcrumbs, internal-linking strategy, XML sitemap and
`robots.txt`. Commercial links use `rel="sponsored nofollow"`; every commercial/review page
carries an affiliate disclosure and a responsible-gambling notice.

## ⚠️ Placeholder data

Casinos (`src/data/casinos.ts`) and providers are **fictional placeholders**. Article bodies
are generated by a topic-aware content engine (`src/lib/content.ts`) as editorial scaffolding.
Replace both with real licensed operators, affiliate links and hand-written copy before going
live.
